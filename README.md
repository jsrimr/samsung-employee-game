# 신입사원 키우기 🏢

**부제: 신입사원 하나를 키우는 데는 온 마을이 필요하다**

삼성전자 신입사원의 성장 스토리를 다룬 시뮬레이션 게임입니다.

## 🎮 게임 소개

평범한 사원에서 시작해 임원까지 승진하고, 차세대 메모리 반도체 HBM4를 개발하여 회사를 시가총액 천조원 기업으로 성장시키는 것이 목표입니다.

## ✨ 게임 특징

### 🎮 게임플레이
- **3가지 스탯 시스템**: 업무능력 💼, 인맥 🤝, 체력 ❤️
- **7개 직급 성장**: 면접 → 사원 → 대리 → 과장 → 차장 → 부장 → 임원
- **30+ 드라마틱한 이벤트**: 각 직급마다 구체적이고 몰입감 있는 스토리
- **HBM 프로젝트**: HBM2 → HBM3 → HBM3E → HBM4 단계별 개발
- **즉시 게임오버**: 스탯 중 하나라도 0 이하가 되면 게임 종료

### 🎨 비주얼 & 사운드
- **레트로 픽셀 아트 스타일**: 포켓몬 골드 감성의 도트 그래픽
- **8bit 칩튠 음악**: Web Audio API로 생성되는 전자음
- **포켓몬 스타일 대화창**: 깜빡이는 화살표, 타이핑 효과
- **부드러운 애니메이션**: 승진 효과, 스탯 바 shimmer

### 📖 몰입감 있는 스토리
- **입사 전 면접**: 곤란한 질문에 답하는 긴장감
- **하이닉스 동기의 조롱**: "삼성은 끝났어" 빌런 캐릭터
- **첫사랑과의 재회**: 일과 사랑 사이의 갈등
- **정리해고 딜레마**: 고교 친구 아버지를 떠올리며...
- **기술적 결단**: GAA vs 3D 패키징, 실제 업계 이슈 반영

## 🎯 게임 목표

1. HBM4 개발 완료
2. 엔비디아 납품 계약 체결
3. 회사 시가총액 천조원 달성

## 🕹️ 플레이 방법

1. 게임 시작 버튼 클릭
2. 각 상황에서 선택지를 읽고 선택
3. 스탯을 관리하며 임원까지 승진
4. HBM4를 완성하고 게임 클리어!

⚠️ **주의**: 업무능력, 인맥, 체력 중 하나라도 0이 되면 게임오버입니다!

## 🚀 플레이하기

[여기를 클릭하여 게임 시작하기](https://jsrimr.github.io/samsung-employee-game/)

또는 `index.html` 파일을 다운로드하여 브라우저에서 직접 실행하세요.

## 💻 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, Keyframe Animations, 픽셀 아트 스타일
- **Vanilla JavaScript (ES6+)**: 객체지향 게임 로직
- **Web Audio API**: 실시간 8bit 음악 및 효과음 생성

## 📱 반응형 디자인

모바일, 태블릿, 데스크톱 모든 기기에서 플레이 가능합니다.
